## ② GPS 검증 (실외)

**코스 준비**: 거리를 아는 코스 1개 — 운동장 트랙 1바퀴(400m) 또는 지도 앱(네이버지도/카카오맵)으로 미리 잰 1km 구간 추천.

1. 코스 시작점에서 [러닝 시작] 탭 → 위치 권한 물으면 **허용**.
2. 화면을 켜둔 채로 그대로 걷거나 뛰며 코스를 완주. (화면을 끄거나 앱을 벗어나면 안 됨 — 이번 빌드는 포그라운드 추적만 지원)
3. 코스 중간에 신호등 등으로 **잠깐 완전히 멈춰서 30초~1분 가만히 서 있어보기** (드리프트 확인용).
4. 완주 후 [일시정지] → [종료·저장].
5. "러닝 기록" 목록에 방금 뛴 기록이 뜨는지 확인.

**합격 기준**:
- 앱에 표시된 최종 거리가 실제 코스 거리 대비 **±5% 이내** (예: 1km 코스면 0.95~1.05km).
- 페이스(분'초"/km)가 실제 체감 속도와 비슷함 (예: 걷기면 10~15분/km, 조깅이면 5~8분/km 근처). "-"나 비정상적으로 빠른 값(예: 0'0x"/km)이 뜨면 안 됨.
- **가만히 서 있던 30초~1분 동안 거리 숫자가 계속 올라가지 않아야 함** (드리프트 없음). 화면에 표시되는 큰 숫자(km)를 서 있는 동안 캡처 2장(시작 시점/1분 후)으로 비교.
- 종료 후 목록에 "GPS 러닝" 배지로 기록 1건이 정상 저장됨.

**실패 시 캡처**:
- 완주 직후 결과 화면(거리·시간·페이스 보이는 화면) 스크린샷 1장 + "실제로는 몇 km/몇 분 코스였는지" 메모.
- 가만히 서 있는 동안 거리가 올랐다면, 서 있기 시작 시점과 끝난 시점의 화면 2장.
- ±5%를 벗어났다면 실제 거리(트랙 바퀴 수 또는 지도 앱 캡처)와 앱 표시 거리를 함께 적어서 전달.

## ③ 워치 검증 (Health Connect + 갤럭시워치)

**사전 준비 (최초 1회)**:
1. Play스토어에서 **"Health Connect"** 앱 설치 (없다면).
2. 삼성헬스 앱 열기 → 설정 → **Health Connect 동기화 켜기** → "운동"·"거리"·"심박수" 항목 **쓰기 허용**.
3. 갤럭시워치가 폰과 연결되어 있고 삼성헬스와 동기화되는지 확인.

**검증 절차**:
1. 갤럭시워치에서 **운동(러닝) 시작** → 원하는 거리만큼 달리기 → 워치에서 **운동 종료·저장**.
2. 폰에서 삼성헬스 앱을 열어 해당 운동이 삼성헬스에 반영됐는지 확인(보통 몇 분 내 동기화됨). 반영 안 됐으면 몇 분 더 기다렸다가 재확인.
3. 모두의 마라톤 앱 "러닝" 화면 → 이름 입력되어 있는지 확인 → **[워치 불러오기]** 탭.
4. **최초 1회 "건강정보 수집 동의" 안내**가 뜸(심박 = 민감정보 별도 동의). 심박까지 저장하려면 **[동의하고 불러오기]**, 거리·시간만 원하면 **[심박 없이 불러오기]** 선택. (검증은 [동의하고 불러오기] 권장)
5. 이어서 Health Connect 권한 팝업(운동·거리·심박수 읽기)이 뜨면 **모두 허용**.
6. "워치 동기화 완료" 알림에 몇 개·몇 km 불러왔는지 표시됨. 목록에 "갤럭시워치" 배지로 기록이 뜨는지 확인.
7. **[워치 불러오기]를 다시 한 번 더 탭** (같은 세션 재확인). 이번엔 동의 안내가 다시 뜨지 않아야 함(1회 동의 저장 확인).

**합격 기준**:
- **최초 [워치 불러오기] 시 "건강정보 수집 동의" 안내가 뜨고, 동의 후 두 번째부터는 다시 뜨지 않음.**
- 목록에 방금 워치로 뛴 기록이 "갤럭시워치" 배지로 1건 저장됨. 거리·심박(♥ 표시)이 삼성헬스 앱에 보이는 값과 비슷함.
- **[워치 불러오기]를 두 번 눌러도 같은 세션이 중복으로 2건 생기지 않고 1건만 유지됨** (핵심 확인 포인트).
- (참고) 삼성헬스·구글핏이 같은 러닝을 각자 기록 중이어도 거리가 **2배로 뻥튀기되지 않아야 함**(단일 출처만 채택). 삼성헬스 표시 거리와 비슷하면 정상.
- **[심박 없이 불러오기]로 불러온 기록엔 심박(♥)이 표시되지 않아야** 함(선택 시 최소수집 확인).
- Health Connect 미설치/미동기화 상태에서 [워치 불러오기]를 누르면 크래시 없이 안내 메시지만 뜸.

**실패 시 캡처**:
- "워치 동기화 완료" 알림 문구 스크린샷.
- 러닝 목록 화면(중복 여부 확인용) — 특히 두 번째 [워치 불러오기] 이후 화면.
- 삼성헬스 앱에 표시된 같은 운동 기록 화면(거리·심박 비교용).
- 실패 메시지가 뜬 경우 그 문구 그대로 캡처.
